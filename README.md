# ☕ Maratona Java - DevDojo

Repositório dedicado aos meus estudos e práticas acompanhando o curso **Maratona Java** do canal DevDojo.

## 📚 Sobre o Repositório

Este repositório contém todos os códigos, exercícios e anotações desenvolvidos durante as aulas. O objetivo é construir uma base sólida em Java, desde os conceitos básicos até os mais avançados.

## 🚀 Módulos e Tópicos Abordados

Abaixo estão os módulos estudados e documentados neste repositório até o momento:

### 🔹 Introdução (`introducao/`)
- Estrutura básica e `Olá Mundo`
- Tipos Primitivos (Declaração e exercícios)
- Operadores (Aritméticos, lógicos, relacionais, etc.)
- Estruturas Condicionais (`if`, `else`, `switch`)
- Estruturas de Repetição (`while`, `do-while`, `for`)
- Arrays Unidimensionais
- Arrays Multidimensionais

### 🔹 Java Core (`JavaCore/`)
- **A. Introdução a Classes:** Criação de classes, atributos, instanciação de objetos.
- **B. Introdução a Métodos:** Criação de métodos, passagem de parâmetros, retorno, `this`.

## 🛠️ Tecnologias e Ferramentas Utilizadas

- **Linguagem:** Java
- **IDE:** IntelliJ IDEA

## 📌 Estrutura de Diretórios

O projeto está organizado em pacotes/diretórios correspondentes a cada módulo do curso, facilitando a navegação:

```text
src/academy/devdojo/maratona/java/
├── introducao/       # Aulas de lógica e fundamentos do Java
└── JavaCore/         # Aulas de Orientação a Objetos e tópicos avançados
    ├── AIntroducaoClasses/
    └── BIntroducaoMetodos/
```
